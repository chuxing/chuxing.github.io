这个理由足够吗？

发表于 2016-05-14   |   分类于 两个人   |     |   阅读次数

世人笑我太疯癫，我笑世人看不穿。
痴痴傻傻为哪般，莽莽撞撞求爱情。
曾经说，最爱你的洒脱，就像那萨拉弗。
为了你，甘愿坠入凡尘。

回答

故事的经过就不想再提了，结果就是我在楼下喊名字， 你发了一条信息给我。我没有回。
“让我觉着曾经的青春没有看错人行吗”。这是你说的话。信息中的其中一句话。因为难过的事不想在这里写，所以其他的内容就不提了。
看了之后，说不上什么感觉。不是生气， 不是郁闷，也不是难过。因为胡闹过后，自己也累了。折腾不动了。一些想说的话，在这里记一下吧。

对不起，最后一次胡闹了。我知道你喜欢成熟，可是有些成熟是装不来的，也不是我这个年龄段该有的。
一些无理智，疯狂，我并不认为有什么不妥。

你说我没有人格魅力也好，说我不要尊严也罢；我接受但不认同，
只有做了我觉着要做，在你看来是胡闹的事情之后，才会知道自己会不会后悔，除了删除微信比较蠢以外，
去火车站，发短信开玩笑，在楼下等你喊你，我都不认为自己做得有多离谱。谈恋爱嘛，
有时候需要些疯狂，需要些无理智。始终如一，一潭死水有什么意思？看着什么事情都好好好，
出了点问题，蹦的就谈分手。微信事件不知道是导火索还是最后的稻草，我解释过了，但你选择漠视。
哦，突然想到一次词“夺命连环call”，我也做过几次吧。是挺蠢的。这个在这里道一个歉。

从高中开始，你可以每天载我上下学，可以默默地喜欢着我，可以追随我的步伐，再到后来，
你独自坐飞机去哈尔滨，告诉我很多秘密。问到我上课的地点去给我惊喜，给我织围巾，一起写小说，
记录点点滴滴，容忍我在别人看来非常混蛋的不打电话，不联系举动。再到最后硕士阶段，
你可以到我们宿舍，一层一层，一个宿舍一个宿舍的找我。而且每次还都要你主动说在一起。
我只是一直在傻傻的“复合吧复合吧”。问个不停，什么也不做。

你为我做了这么多，在我看来，值得我放下尊严，不要什么人格魅力，去挽回你。在我看来，
你很善良，很完美，很漂亮（来自一个超级外貌协会典型水瓶座的衷心赞美），
失去过一次之后我一直都太害怕再次失去了，越害怕越想付出，越付出有时候越会出错，
而忘记了之前是为什么失去你的。一次又一次，无声的诉说。

都说失去了才懂得珍惜。一开始我是真的很珍惜，珍惜到你说什么我就信，不想让你困扰，
结果呢？天天想你，憋在心里，在你看来我是不在乎你。
有时候我真的很恨这个命运，还有这个缘分，屎一样的安排，但我又安慰自己，好事多磨，好事多磨。
真应了那句话，在错的时间遇见对的人，是一种无奈。

那这次呢？我知道你很烦，工作受挫，论文烦心。记得和cmy在假期聊天的时候他说
“在学校好找对象，等毕了业太难了，你等她毕业，没什么选择了再追追她，肯定好追”。
是啊，我也可以等，现在什么都不做，等你工作了，稳定了再去找你。可能这样机会更大一些。
但是在我看来，这样的爱情不是我想要的，我想参与到你的喜怒哀乐中，即使颇有微词，我也可以接受。
而不是不闻不问，什么也不说，什么也不做，只是等到时间合适了再去打扰你。
不过，现在我的做法让你困扰了。也让我知道了，可能有时候不做比做更有效。

有时我也在想，这样的结局，以后还有可能吗？但是不做，不说，永远也不知道结果是什么。
这样也能给自己一个理由，给自己一个放弃的理由。可是这些理由，
都抵不过今后的日子里有个能陪伴自己走过一生的人。

记得曾经我说“咱们的主要问题就是在于长时间不在一起，对于彼此的需要不能第一时间知晓，满足。
但是咱们每次在一起的时候都是很和谐的，那么以后在一起生活了，肯定还是每天在一起的时间多啊，
所以只需要熬过了这些困难的日子”。你没有反对，相信你也是赞同的。
彼此的气场的契合，生活习惯，价值观人生观的一致，都是最坚定的基础。
要不然，我也知道强扭的瓜不甜，不会多次去挽回的。
在我看来，现在的一些分歧，就像是一道坎，也许过得去，但也许，我们也会跨不过去。

诚然，很多地方我不了解你，正如我也不了解我自己一样，生活嘛，有摩擦才是常态。

一起坐摩天轮，一起去自然博物馆，一起打羽毛球，一起打网球，教你学游泳，一起去爬一座又一座山峰;
一个又一个曾经许过的诺言。真的想把它们都实现。

叶子的离去到底是因为风的追求，还是树的不挽留。

最后，虽然我不明白，你说“没有看错人”到底是指的我的人品，我的学识，我的精神层次，
我的人生高度的哪个方面，
还是单纯说我太死皮赖脸了。不管是什么，我想，我不会让你失望的。
#胡闹 #尊严 #纪实
